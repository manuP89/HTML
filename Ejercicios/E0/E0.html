Lista de conocimientos

Aqui vamos a enumerar las distintas etiquetas de un documento HTML.
Se enumerarán como una lista:

  1. Head
    • Title
    • Meta
    • Style
    • Link
  2. Body
    • h[x]
      • Define un título
    • p
      • Define un párrafo
    • ol
      • Define una lista ordenada
    • ul
      • Define una lista no ordenada
